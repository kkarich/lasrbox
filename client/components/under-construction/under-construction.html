I know it is hard to tell but, by tapping my phone I am controlling a laser pointer that is mounted on a pan tilt servo kit

How it works

1. The Video

Establish a connection from the recording device to the display device using peer.js

2. User input

Update the laser location by simply tapping the screen or moving the mouse to send a post request to the cattoy api

3. Update laser location

Retrieve and update the laser location by calling the cattoy api then update the servos' X and Y coordinates
